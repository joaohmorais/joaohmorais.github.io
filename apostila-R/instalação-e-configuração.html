2 Instalação e Configuração

Neste capítulo, vamos aprender como instalar o R nos principais sistemas operacionais: Windows, Linux e Mac. Também mostramos a configuração de seu principal ambiente de desenvolvimento: o RStudio.

2.1 Instalação do R

2.1.1 Instalação no Windows

2.1.1.1 Passo 1: Baixar o instalador

  1. Acesse a página do CRAN: CRAN R Project.
  2. Clique na opção “CRAN”, abaixo de “Download”.
  3. Escolha um servidor (mirror) mais próximo de você.
  4. Clique em “Download R for Windows”.
  5. Clique em “base” e depois em “Download R x.y.z for Windows” (onde x.y.z é a versão mais recente).

2.1.1.2 Passo 2: Executar o instalador

  1. Execute o arquivo baixado (R-x.y.z-win.exe).
  2. Siga as instruções do instalador. Recomendamos manter as configurações padrão.
  3. Após a conclusão, o R estará instalado no seu sistema.

2.1.2 Instalação no Linux

Para diferentes distribuições Linux, os passos podem variar. Aqui vamos usar o Ubuntu como exemplo:

  1. Acesse a página do CRAN: CRAN R Project.
  2. Clique em “Download R for Linux”.
  3. Agora selecione a sua distribuição de sistema. Nesse caso, será a “Ubuntu”.
  4. Siga as instruções mostradas na página.

2.1.3 Instalação no MacOS

2.1.3.1 Passo 1: Baixar o instalador

  1. Acesse a página do CRAN: CRAN R Project.
  2. Clique na opção “CRAN”, abaixo de “Download”.
  3. Escolha um servidor (mirror) mais próximo de você.
  4. Clique em “Download R for (Mac) OS X.”.
  5. Clique em “base” e depois em selecione a versão mais recente.

2.1.3.2 Passo 2: Executar o instalador

  1. Execute o arquivo baixado (R-x.y.z.pkg).
  2. Siga as instruções do instalador. Recomendamos manter as configurações padrão.
  3. Após a conclusão, o R estará instalado no seu sistema.

2.2 Instalação do RStudio (posit)

O RStudio é um Ambiente de Desenvolvimento Integrado (Integrated Development Environment - IDE), que auxilia a escrita de códigos e desenvolvimento de projetos em R.

  • Para instalá-lo, acesse o website da Posit, na seção de Downloads: Downloads - RStudio;
  • A partir daí, escolha o arquivo que corresponde ao seu sistema operacional (Windows, Linux, MacOS, …);
  • Siga as instruções de instalação.
